Personal Data Processing Policy

1. General Provisions

This Personal Data Processing Policy (hereinafter referred to as the "Policy") defines the procedure and conditions for processing personal data undertaken by the Zelux Spark blog administration (hereinafter referred to as the "Operator") and establishes measures to ensure the security of this data.

1.1. The Operator considers the observance of human and civil rights and freedoms when processing personal data, including the protection of rights to privacy and personal secrets, to be the most important goal and condition for its activities.

1.2. This Policy applies to all information that the Operator may obtain about visitors (hereinafter referred to as "Users") of the zelux-spark.blog website.

2. Basic Concepts Used in the Policy

  • Website — a collection of graphic and information materials, as well as computer programs and databases that ensure their availability on the Internet at the network address zelux-spark.blog.
  • User — any visitor to the website.
  • Personal Data — any information relating directly or indirectly to a specific or identifiable User of the website.
  • Processing of Personal Data — any action (operation) or set of actions (operations) performed with personal data, including collection, recording, systematization, storage, updates, extraction, use, transfer, depersonalization, blocking, and destruction.
  • Personal Data Security — protection of personal data from unauthorized or accidental access, destruction, modification, or copying.
  • Consent — a voluntary, informed, and conscious expression of the User's will to process their personal data.

3. Data We Process

The Operator may process the following categories of data:

  • Personal data provided by the User:
    • Name (or nickname);
    • Email address;
    • Phone number (when filling out feedback forms).
  • Automatically collected data:
    • The site collects and processes anonymized data about visitors (including "cookies") using internet statistics services (Google Analytics and others). This data includes IP address, browser type, and access time.

The above-mentioned data are hereinafter collectively referred to as Personal Data in this Policy.

4. Purposes of Processing Personal Data

  • Identification of the User to provide access to the blog's educational content.
  • Establishing feedback with the User, including sending notifications and requests regarding the use of AI technologies and processing User applications.
  • Informing the User by sending emails. The User can always opt-out of receiving messages by sending an email to support@zelux-spark.blog marked "Unsubscribe from notifications".
  • Improving website performance and analyzing audience interests in bot development.

5. Legal Grounds for Processing Personal Data

The Operator processes the User's personal data only on legal grounds:

  • With the voluntary consent of the User, expressed by filling out forms on the website or checking the corresponding box.
  • Processing of anonymized data is carried out if permitted in the User's browser settings (cookie saving is enabled).

6. Collection, Storage, and Data Security

  • The Operator ensures the safety of personal data and takes all possible technical measures to protect data from unauthorized access.
  • The User's personal data will never be transferred to third parties, except in cases provided for by applicable law.
  • The User can update their data or withdraw consent for its processing by sending a notification to the email address support@zelux-spark.blog.

7. Final Provisions

  • The User can obtain clarification on data processing issues by contacting the Operator via email at support@zelux-spark.blog.
  • The current version of the Policy is freely available on the Internet at zelux-spark.blog.
